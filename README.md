> **Udacity Self-Driving Car Engineer Nanodegree** -- PID Control

PID controller implementation for vehicle steering and speed control in a driving simulator, with manual and Twiddle-based parameter tuning.

---

## Key Concepts

- Proportional-Integral-Derivative control
- Cross-track error minimization
- Twiddle optimization algorithm
- Steering and throttle control
- Parameter tuning strategies

---

## Getting Started

```bash
mkdir build && cd build
cmake .. && make
./pid
```
